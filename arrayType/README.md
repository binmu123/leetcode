该文件夹下的算法题主要是数组类题目
LongestConsecutive
   最长连续字符（原字符串的字符不相邻）
RemoveDuplicates
   存在两个方法（算法），I、去除array重复出现的字符
                         II、去除array重复出现2次的字符 
                               扩展如果为n次呢？采用何种方法
TwoSum
   存在两个方法，I、查询数组中是否存在两数之和为target
                 II、是否存在三个数之和为target 
                        扩展如果存在n个数和为target，如何求？
